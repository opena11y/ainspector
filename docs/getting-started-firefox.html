Firefox Browser

Edge Add-ons

  • To install the extension first go to the Mozilla Add-ons website.
  • Use the "Add to Firefox" button to install extension into the Edge browser.
  • The icon will initially be hidden, but can be pinned to the toolbar using the extension manager.
Show Hide Screen Shot

Pinning to Toolbar

After you install AInspector for WCAG Evaluation you can pin the icon to the toolbar to open and close the side panel.

  • Select the extensions manager icon in the Firefox toolbar and locate AInspector for WCAG Evaluation in the list of extensions.
  • Use the "Pin to Toolbar" toggle button in the popup menu to the extension to enable icon to appear in the toolbar.
  • Selecting the toolbar icon opens and closes the side panel.
Show Hide Screen Shot

Keyboard Shortcut

A keyboard shortcut is predefined in Firefox for opening and closing the side panel.

  • Win/Linux: ctrl+shift+1
  • Mac: cmd+shift+1

Changing Side Panel Position

The side panel by default is positioned on the left side of the screen and cannot be configured for the left side.

Privacy

AInspector for WCAG Evaluation does not collect or store any information about users or work with any other parties to collect or share user browsing information.

Feedback

If you find a problem with AInspector for WCAG Evaluation please let us know by creating an issue in the GitHib repository for AInspector.  Please provide the following information:

  1. A URL to were the problem can be replicated.
  2. A short description of the issue.
  3. How we can contact you for more information and verification the issue is resolved.
