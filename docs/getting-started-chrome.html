Chrome Browser

Chrome Webstore

  • To install the extension first go to the chrome web store.
  • Use the "Add to Chrome" button to install extension into Chrome or chrome compatible browser.
  • The icon will initially be hidden, but can be pinned to the toolbar using the extension manager.
Show Hide Screen Shot

Pinning to Toolbar

After you install AInspector for WCAG Evaluation you can pin an icon to the toolbar to open and close the side panel.

  • Select the icon in the toolbar and locate the in the list of extensions.
  • After the extension name there is a toggle button to enable icon to appear in the toolbar.
  • Selecting the toolbar icon opens and closes the side panel.
Show Hide Screen Shot

Setting a Keyboard Shortcut

A keyboard shortcut can be assigned to open and close the side panel.

  • Go to the Manage Extensions settings page (e.g. URL: chrome://extensions/).
  • Select the keyboard shortcuts link (e.g. URL: chrome://extensions/shortcuts).
  • Select the edit icon associated with the extension and focus moves to the edit box.
  • Type the key combination for the keyboard shortcut, for example Control+Shift+A.
  • When viewing a web page the shortcut can be used to open and close the side panel.
Show Hide Screen Shot

Changing Side Panel Position

The side panel by default is positioned on the right side of the screen. It can be changed to the left side in settings.

  • Open the setting options by going to the Chrome option in the main menu and selecting the Settings option.
  • Use the search box with the value "side panel" to filter the settings.
  • Go to the "Side panel position" option and you can choose between: "Show on right" or "Show on left" options.
