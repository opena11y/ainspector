Rule Group View

Features

Feature Description
Group Title The group title is the name of the Rule Category or WCAG Guideline associated with the list of rule results in the Rule Results Grid.
Results Summary Provides a count of the number of rules with Violations (V), Warnings (W), Manual Checks (MC) and Passed (P) for the rules in the rule group.
Rule Results Grid The Rule Results Grid has four columns:
  1. Rule summary
  2. Result (i.e. Violation, Warning, Manual Check or Passed)
  3. WCAG Success Criteria
  4. WCAG Level (i.e. A, AA, AAA)
  5. Required for WCAG Conformance (i.e. Yes or No)
Rule Information Rule Information for the selected rule in the Rule Results Grid include the following properties:
  • Rule Id
  • Rule Scope (i.e. element, page or website)
  • Summary
  • Definition
  • Action
  • Purpose
  • Techniques
  • Target ELements
  • Primary WCAG Compliance
  • Other Related WCAG Success Criteria
  • Additional Information
Evaluation Information This section appears in every view and includes information on:
  • Page title
  • URL of the web page
  • The selected Ruleset
  • Scope of rules used in the results (e.g. All, Page, Website)

Buttons

Button Description
Back The Back button changes the view back to the Summary view.
Copy The Copy button copies the Selected Rule information to the clipboard in a plain text format.
Details The Details button changes the view to the rule result associated with the selected rule in the rule group results grid.
Export Data The Export Data button prompts to save the rule group result data to a local file system in CSV or JSON format. The Export button is available in all views.
Options The Options button opens option pages for the extension. The Options button is available in all views
Rerun Evaluation The Rerun Evaluation button prompts the user to rerun the evaluation on the current page. The Rerun button is available in all views.
Views The Views button menu opens a menu to navigate to one of the rule group views and is an alternative to using the grid and details button. Handy for keyboard users, especially when shortcuts are enabled. The Views button is available in all views.
