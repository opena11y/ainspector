Summary View

Features

Feature Description
Summary Title The summary title provides information of the rules include in the evaluation result.

Values include:
  • All Rules: All rules are included in evaluation results.
  • Page Rules: Only rules with rule scope of page are included in evaluation results.
  • Website Rules: Only rules with rule scope of website are included in evaluation results.
  • First Step Rules: Only rules in the First Step rule set are includes in the evaluation result.
Results Summary Provides a count of the number of rules with Violations (V), Warnings (W), Manual Checks (MC) and Passed (P).
Rule Groups Grid

The Rule Group Grid has two tabs: Rule Category and WCAG Guidelines. Choosing a tab shows rule groups of rules based on the tab selected.

The Rule Groups Grid has five columns:
  1. Rule group title
  2. Number of rules with violations
  3. Number of rules with warnings
  4. Number of rules with manual checks
  5. Number of rules with pass.

The rule groups grid helps identify which features are meeting accessibility requirements and what areas will require human inspection. Each rule is associated with both a Rule Category and WCAG Guideline. The rule categories use labels familiar to designers and developers that have an understanding of html coding. The WCAG Guidelines are part of the standards used to define accessibility and are familiar to accessibility specialists.

Evaluation Information This section appears in every view and includes information on:
  • Page title
  • URL of the web page
  • The selected Ruleset
  • Scope of rules used in the results (e.g. All, Page, Website)

Buttons

Button Description
Details The Details button changes the view to the selected Rule Category or WCAG Guidelines rule group.
Export Data The Export Data button prompts to save the summary data to a local file system in CSV or JSON format. The Export button is available in all views.
Options The Options button opens option pages for the extension. The Options button is available in all views
Rerun Evaluation The Rerun Evaluation button prompts the user to rerun the evaluation on the current page. The Rerun button is available in all views.
Views The Views button menu opens a menu to navigate to one of the rule group views and is an alternative to using the grid and details button. Handy for keyboard users, especially when shortcuts are enabled. The Views button is available in all views.
